Selamat datang di CSS Eksternal

halaman ini menggunakan CSSeksternal untuk styling

kenapa CSS eksternal?

  • memisahkan struktur (HTML) dan presentasi
  • CSS dapat digunakan di multiple halaman
  • lebih mudah maintenance
  • file HTML lebih bersih dan readable

Cara menggunakan CSS

Gunakan tag <link> didalam <head>

format: <link rel=stylesheet" href="latihan1.css">
